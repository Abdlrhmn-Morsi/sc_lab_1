1 - git init : Initializes a new Git repository in the current directory.

2 - git clone : Creates a local copy of a remote repository.

3 - git status : Displays the status of the working directory and staging area. It shows which files are modified, staged, or untracked.

4 - git add : Stages changes to a file, preparing it for a commit..

5 - git commit -m "[message]" : Commits the staged changes to the repository with a descriptive message. This message should summarize the changes made.

6 - git pull : Fetches changes from a remote repository and merges them into the local branch. 

7 - git push : Uploads local commits to a remote repository. 

8 - git branch : Lists all the branches in your repository. 

9 - git checkout [branch] : Switches to a different branch in the repository.

10 - git merge [branch] : Merges the specified branch into the current branch.
